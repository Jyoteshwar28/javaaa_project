# CCRM Usage Guide

## Adding Students
1. Run the program
2. Choose `1` for "Add Student"
3. Enter ID, RegNo, Name, Email

## Adding Courses
1. Choose `2` for "Add Course"
2. Enter Code, Title, Credits

## Enrollment & Grades
1. Choose `3` for "Enroll & Grade"
2. Enter Student RegNo, Course Code, Marks
3. Grade is calculated automatically

## Print Transcripts
- Choose `4` to print all student transcripts

## Import CSV
- Choose `5`
- Enter file paths for `students.csv` and `courses.csv`

## Export CSV
- Choose `6`
- Files exported to `students_export.csv` & `courses_export.csv`

## Backup
- Choose `7`
- Program copies all files to `ccrm-data/backup_TIMESTAMP/`  
- Recursive size is printed

## Reports
- Choose `8`
- GPA distribution & top 5 students displayed

## Exit
- Choose `0`
